Virtual Keyboard Alt-Keys Popup Demo

Note: Look at this demo source or refer to the documentation on how to set up this extension.


  • NOTE In keyboard core v1.28.2, this extension has switched to using event.key which is only supported in modern browsers. Use an extension version prior to v2.0.0 (not the core version) to support older browsers.

  • Click or press and hold any of the following keys to see a popup:
    • Extension defaults: 1234567890acdeghijklnorstuwyz'ACDEGHIJKLNORSTUWYZ!$()"?
    • Demo extras: [, ], enter, ☺ & ✙ (examine the source code of this page to see how these were added).
  • The keys within the popup are completely customizable. In fact they can include any action key (click or press and hold the enter key).
  • In v1.25.11, it is possible to use the physical keyboard. Press & hold an assigned key for the popup to open, then:
    • Use left arrow (←), right arrow (→), Home, End to navigate.
    • Press Enter to insert the highlighted key.
    • Press escape (Esc) to close the popup.
